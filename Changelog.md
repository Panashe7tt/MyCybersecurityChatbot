# Changelog

## [1.0.0] - 2026-06-27

### Added
- Chat interface with keyword-based responses
- Sentiment detection for user input
- 12-question cybersecurity quiz
- Task manager with reminders
- Activity logging system
- Audio greeting on startup
- JSON data persistence
- Sidebar navigation
- Typing indicator for chat responses

### Features Detail
- NLP simulation with keyword matching
- Support for natural language reminders (tomorrow, next week)
- Chat history saved locally
- Tasks saved in Documents folder
- Quiz with instant feedback and explanations

### Technical
- .NET 8.0 WPF application
- Newtonsoft.Json for data storage
- System.Media for audio playback

## [0.9.0] - 2026-06-20

### Added
- Initial project setup
- Basic chat functionality
- Quiz question framework

### Known Issues
- Audio playback requires .wav file in output directory
- Reminder feature only supports English language keywords
