# My Cybersecurity Chatbot

A Windows desktop application built with C# and WPF that helps users learn about cybersecurity through interactive chat, quizzes, and task management.

---

## Features

### Smart Chat System
- Keyword-based responses for cybersecurity topics
- Sentiment analysis to detect user emotions
- Typing indicator for realistic chat experience
- Audio greeting on startup
- Chat history saved locally

### Cybersecurity Quiz
- 12 questions covering:
  - Phishing attacks
  - Password security
  - 2FA and MFA
  - Ransomware
  - Social Engineering
  - GDPR
  - Backup strategies
  - Public Wi-Fi safety
- Instant feedback with explanations
- Final score tracking

### Task Manager
- Add tasks with titles and descriptions
- Natural language reminders (e.g., "tomorrow")
- Mark tasks as complete/incomplete
- Persistent storage in your Documents folder

### Activity Log
- Track all user actions
- View last 10 activities with timestamps

---

## Technologies Used
- Framework: WPF (.NET 8.0)
- Language: C# 8.0
- UI: XAML
- Data Storage: JSON (Newtonsoft.Json)
- Audio: System.Media.SoundPlayer
