# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in this application, please report it by:

1. Opening an issue on GitHub
2. Describing the vulnerability in detail
3. Providing steps to reproduce

## Security Best Practices

This application follows these security practices:

- All user data is stored locally (no data sent to external servers)
- No sensitive information is logged
- JSON data files are stored in the user's Documents folder
- Input validation is performed on all user inputs
- No external API calls are made

## Data Privacy

- Chat messages are stored locally in chat.json
- Tasks are stored locally in CyberTasks.json
- No data is shared with third parties
- Users can delete data by removing the JSON files

## Recommendations for Users

- Keep the application updated
- Do not share your Documents folder with untrusted users
- Regularly backup your chat and task data
